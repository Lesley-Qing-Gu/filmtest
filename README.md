# CineRound | 银幕之外

一个致力于影评、电影节报道与深度访谈的电影评论平台。

## 在线预览

🔗 https://p1ge0n8.github.io/repo_test/

## 功能

- 最新资讯 / 电影节报道 / 深度文章 / 访谈
- 文章详情页（支持 Markdown 渲染）
- 管理后台（文章的增删改）

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000
